# 蓦然回首韩文版-翻译

![71d891b48b31edefbeb511fd9f70043e.jpeg](71d891b48b31edefbeb511fd9f70043e.jpeg)

이번 주 학년 신문을 나눠 주겠다~

> 我要发这周的年级报啦~
> 

이번 주 4컷 만화도 엄~청 재미 있었어!

> 这周的四格漫画也超级有意思!
> 

![a92c89c6ca76007a1b06633478b9c707.jpeg](a92c89c6ca76007a1b06633478b9c707.jpeg)

- **第一格：**
    - 背景文字：둘 다 사고로 죽기 직전（两人在事故中濒死之际）
    
    > 
    > 
    > 
    > **기 직전 (就在……之前 / 濒临……之际)**
    > 
    > - **语法公式**：`动词词干 + 기 직전`
    > - **场景还原**：`죽다` (死) + `기 직전` = **濒死之际 / 就在要死掉之前**。
    > - **反义词**：`기 직후` (就在……之后)。
    - 左侧气泡：약속할게...（我答应你...）
    - 右侧气泡：다시 태어나도 또 나랑 키스해 줘（就算转世重生，也要再和我接吻哦）
    
    > 
    > 
    > 
    > **태어나다（出生、诞生）**
    > 
    > - **태어 (tae-eo)**：源自“胎 (태)”，带有从母体中出来的意思。
    > - **나다 (na-da)**：是一个万能动词，表示“出来、产生、发生”。
    > - **合起来：** “出生”、“诞生”。
- **第三格：**
    - 上方方框：20년 후（20年后）
    - 左侧气泡：당신은 어딨어?（你在哪里？）
    - 右侧气泡：다시 태어났어!!（我重生啦！！）
- **第四格：**
    - 画面标注：지구（地球）/ 운석（陨石）

실은 이거, 좀 바빠서 5분 만에 그렸는데 말이지~

> 有点忙，所以是5分钟就画出来的啦~
> 

| **韩文原文** | **语法解析** |
| --- | --- |
| **실은** | 当你想纠正对方的误解，或者准备说出一个对方不知道的“真相”、“秘密”时，用它作为句子的开头。 |
| **5분 만** | `만` 是一个**依存名词**，接在时间词后面，表示“**（经历了多少时间）之后**”或者“**在多久的时间内**”。 |
| **그렸는데** | 原型：画画 그리다  过去时(动/形)：词干+-았/었/였는데 |
| **말이지~** | “嘛.../不是嘛...”。一种拽拽的、寻求对方关注的语气结尾。 |

그런 것치고는 그럭저럭 괜찮게 그려졌다는 느낌이랄까?

> 就那种情况（只花了5分钟画出来）来说，感觉画得还算挺不错的吧？／怎么说呢，好像有种“其实还可以”的感觉？
> 

| **韩文原文** | **对应发音** | **含义解析** |
| --- | --- | --- |
| **그런 것** | Geu-reon geot | 这种程度、那样的情况（指代只画了5分钟这件事）。 |
| **치고는** | Chi-go-neun | **（语法重点）** 考虑到……、就……而言。 |
| **그럭저럭** | Geu-reok-jeo-reok | 凑合、还算、马马虎虎。 |
| **괜찮게** | Gwaen-chan-ke | 挺好地、不错地（形容词 괜찮다 的副词形式）。 |
| **그려졌다** | Geu-ryeo-jjeot-da | 被画出来（그리다 的被动型 + 过去时）。 |
| **느낌이랄까?** | Neu-kkim-i-ral-kka? | (이)라고 할까的缩写，要说是……吗？ |

학년 신문의 4컷 만화 코너 말인데.

> 说的是年级报纸的四格漫画版块嘛。

| **韩文原文** | **语法解析** |
| --- | --- |
| **학년 신문의** | 학년（年级）+ 신문（报纸）+ 의（的），"年级报纸的" |
| **4컷 만화 코너** | 4컷（四格）+ 만화（漫画）+ 코너（栏目/版块），外来语 corner |
| **말인데** | 말이다（说的是……）+ -ㄴ데，用于引出话题，相当于"说起……"、"关于……嘛" |

---

한 줄만 쿄모토에게 내줘도 되겠니?

> 能不能把一个版块让给京元呢？

| **韩文原文** | **语法解析** |
| --- | --- |
| **한 줄만** | 한（一）+ 줄（行/排/栏）+ 만（只），这里指"一个版块" |
| **쿄모토에게** | 쿄모토（京元，人名）+ 에게（给……/对……），方向格助词 |
| **내줘도** | 내주다（让出、交出）的 -아/어도 形式，"就算让出……" |
| **되겠니?** | 되다（可以）+ -겠-（推测/委婉语气）+ -니（疑问终结词），"可以吗？" |

---

뭐. 전 딱히 상관없지만요.

> 啥。我倒是没什么特别意见就是了。

| **韩文原文** | **语法解析** |
| --- | --- |
| **뭐** | 语气词，表示"没什么/随便"，带有漫不经心的语感 |
| **전** | 저는 的缩略，"我（是）" |
| **딱히** | 副词，"特别地、明确地"，常搭配否定，表示"也没什么特别的" |
| **상관없다** | 没关系、无所谓 |
| **-지만요** | 表示转折的终结语尾，带礼貌感，"……就是了/……不过嘛" |

옆 반의 등교 거부아인데.

> 是隔壁班拒绝上学的那个孩子。

| **韩文原文** | **语法解析** |
| --- | --- |
| **옆 반의** | 옆（旁边/隔壁）+ 반（班）+ 의（的） |
| **등교 거부아** | 등교（上学）+ 거부（拒绝）+ 아（孩子，아이的口语缩略），"拒绝上学的孩子" |
| **-인데** | 이다（是）+ -ㄴ데，说明背景，"是……嘛/是……的" |

학교는 못 나와도 만화는 그린다는 것 같더구나.

> 虽然不来上学，但好像还是在画漫画呢。

| **韩文原文** | **语法解析** |
| --- | --- |
| **학교는 못 나와도** | 못（不能）+ 나오다（出来/来）+ -아도（即使……），"就算不来上学" |
| **그린다는** | 그리다（画）+ -ㄴ다는（引用形式），"说是在画/画着的" |
| **것 같더구나** | 것 같다（好像/似乎）+ -더구나（说话者对新发现的感叹），"好像是这样呢" |

봐 줄 만한 그림을 그리는 거, 아마추어한테는 어려운 일이에요.

> 画出值得一看的画，对业余爱好者来说是件难事。

| **韩文原文** | **语法解析** |
| --- | --- |
| **봐 줄 만한** | 봐주다（欣赏/看）+ -ㄹ 만하다（值得……），"值得一看的" |
| **아마추어한테는** | 아마추어（业余爱好者，外来语）+ 한테는（对……来说），口语化方向格 + 对比助词 |
| **어려운 일이에요** | 어렵다（难）+ -ㄴ（修饰语尾）+ 일（事情）+ 이에요（是），"是一件难事" |

---

학교에도 못 오는 나약한 애가 만화를 그릴 수 있을까요?

> 连学校都来不了的软弱孩子，能画漫画吗？

| **韩文原文** | **语法解析** |
| --- | --- |
| **학교에도 못 오는** | 학교（学校）+ 에도（连……都）+ 못（不能）+ 오다（来）+ -는（现在时定语型），"连学校都来不了的" |
| **나약한 애** | 나약하다（软弱的）+ -ㄴ（修饰）+ 애（孩子，아이 的口语缩略） |
| **그릴 수 있을까요?** | 그리다（画）+ -ㄹ 수 있다（能够……）+ -ㄹ까요（疑问/揣测终结词），"能画吗？" |

학년 신문 나눠 주마.

> 我来把年级报纸发给大家。

| **韩文原文** | **语法解析** |
| --- | --- |
| **나눠 주마** | 나누다（分发）+ -아/어 주다（为他人做）+ **-마**（终结词尾，表说话者的承诺/意志，多用于长辈对晚辈），"我来给你们发" |

부모님께도 보여 드리고~.

> 也给爸妈看看~。

| **韩文原文** | **语法解析** |
| --- | --- |
| **부모님께도** | 부모님（父母，含尊称 -님）+ 께（对长辈用的方向格助词，에게 的敬语）+ 도（也） |
| **보여 드리고~** | 보이다（展示/让看）+ -어 드리다（为长辈做某事的敬语形式）+ -고（并列/延续），"也让（家长）看看~" |
