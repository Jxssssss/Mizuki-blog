- **第一格：**
    - 背景文字：둘 다 사고로 죽기 직전（两人在事故中濒死之际）
    
    > 
    > 
    > 
    > **기 직전 (就在……之前 / 濒临……之际)**
    > 
    > - **语法公式**：`动词词干 + 기 직전`
    > - **场景还原**：`죽다` (死) + `기 직전` = **濒死之际 / 就在要死掉之前**。
    > - **反义词**：`기 직후` (就在……之后)。
    - 左侧气泡：약속할게...（我答应你...）
    - 右侧气泡：다시 태어나도 또 나랑 키스해 줘（就算转世重生，也要再和我接吻哦）
    
    > 
    > 
    > 
    > **태어나다（出生、诞生）**
    > 
    > - **태어 (tae-eo)**：源自“胎 (태)”，带有从母体中出来的意思。
    > - **나다 (na-da)**：是一个万能动词，表示“出来、产生、发生”。
    > - **合起来：** “出生”、“诞生”。
- **第三格：**
    - 上方方框：20년 후（20年后）
    - 左侧气泡：당신은 어딨어?（你在哪里？）
    - 右侧气泡：다시 태어났어!!（我重生啦！！）
- **第四格：**
    - 画面标注：지구（地球）/ 운석（陨石）

실은 이거, 좀 바빠서 5분 만에 그렸는데 말이지~

> 有点忙，所以是5分钟就画出来的啦~
> 

| **韩文原文** | **语法解析** |
| --- | --- |
| **실은** | 当你想纠正对方的误解，或者准备说出一个对方不知道的“真相”、“秘密”时，用它作为句子的开头。 |
| **5분 만** | `만` 是一个**依存名词**，接在时间词后面，表示“**（经历了多少时间）之后**”或者“**在多久的时间内**”。 |
| **그렸는데** | 原型：画画 그리다  过去时(动/形)：词干+-았/었/였는데 |
| **말이지~** | “嘛.../不是嘛...”。一种拽拽的、寻求对方关注的语气结尾。 |

그런 것치고는 그럭저럭 괜찮게 그려졌다는 느낌이랄까?

> 就那种情况（只花了5分钟画出来）来说，感觉画得还算挺不错的吧？／怎么说呢，好像有种“其实还可以”的感觉？
> 

| **韩文原文** | **对应发音** | **含义解析** |
| --- | --- | --- |
| **그런 것** | Geu-reon geot | 这种程度、那样的情况（指代只画了5分钟这件事）。 |
| **치고는** | Chi-go-neun | **（语法重点）** 考虑到……、就……而言。 |
| **그럭저럭** | Geu-reok-jeo-reok | 凑合、还算、马马虎虎。 |
| **괜찮게** | Gwaen-chan-ke | 挺好地、不错地（形容词 괜찮다 的副词形式）。 |
| **그려졌다** | Geu-ryeo-jjeot-da | 被画出来（그리다 的被动型 + 过去时）。 |
| **느낌이랄까?** | Neu-kkim-i-ral-kka? | (이)라고 할까的缩写，要说是……吗？ |

| --- | --- |

| **4컷 만화 코너** | 4컷（四格）+ 만화（漫画）+ 코너（栏目/版块），外来语 corner |
| **말인데** | 말이다（说的是……）+ -ㄴ데，用于引出话题，相当于"说起……"、"关于……嘛" |

| **韩文原文** | **语法解析** |
| --- | --- |
| **한 줄만** | 한（一）+ 줄（行/排/栏）+ 만（只），这里指"一个版块" |
| **쿄모토에게** | 쿄모토（京元，人名）+ 에게（给……/对……），方向格助词 |
| **내줘도** | 내주다（让出、交出）的 -아/어도 形式，"就算让出……" |
| **되겠니?** | 되다（可以）+ -겠-（推测/委婉语气）+ -니（疑问终结词），"可以吗？" |
